## 🚀 Кроки для підключення Strapi Cloud

### Крок 1: Завантажити на GitHub

```bash
cd tanu-strapi-backend
git init
git add .
git commit -m "Initial Strapi project for Tanu"

# Створіть новий репозиторій на GitHub і підключіть:
git remote add origin https://github.com/ВАШ_USERNAME/tanu-strapi-backend.git
git branch -M main
git push -u origin main
```

### Крок 2: Підключити до Strapi Cloud

1. Відкрийте [strapi.cloud](https://strapi.cloud)
2. Натисніть **"New Project"**
3. Підключіть свій GitHub акаунт
4. Оберіть репозиторій `tanu-strapi-backend`
5. Натисніть **"Deploy"**
6. Зачекайте 2-3 хвилини поки проект розгорнеться

### Крок 3: Отримати URL та API токен

Після деплою:

1. **Скопіюйте Project URL** (наприклад: `https://tanu-xxx.strapiapp.com`)

2. **Створіть API токен:**
   - Зайдіть в адмінку: `https://tanu-xxx.strapiapp.com/admin`
   - Створіть адмін акаунт (це буде ваш головний акаунт)
   - Перейдіть: **Settings** → **Global Settings** → **API Tokens**
   - Натисніть **"Create new API Token"**
   - Назва: `Tanu Frontend`
   - Тип: **Read-only** (тільки читання)
   - Скопіюйте токен (виглядає як довгий рядок символів)

### Крок 4: Оновити frontend

Відкрийте файл `index.js` у вашому проекті та замініть:

```javascript
var CONFIG = {
  STRAPI_BASE: 'https://tanu-xxx.strapiapp.com',     // ← Вставте ваш URL
  STRAPI_PATH: '/api/products?populate=category,image',
  API_TOKEN: 'eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...',  // ← Вставте ваш токен
  FETCH_TIMEOUT_MS: 5000
};
```

### Крок 5: Завантажити картинки продуктів

1. Зайдіть в адмінку Strapi: `https://tanu-xxx.strapiapp.com/admin`
2. Перейдіть: **Content Manager** → **Products**
3. Відкрийте будь-який продукт
4. Натисніть на поле **Image** → **Upload files**
5. Завантажте картинку з папки `Tanu.hyperesources/`
6. Натисніть **Save** та **Publish**

Повторіть для всіх продуктів.

## 👥 Як дати доступ співробітнику

1. В адмінці Strapi перейдіть: **Settings** → **Administration Panel** → **Users**
2. Натисніть **"Invite new user"**
3. Введіть email співробітника
4. Оберіть роль: **Editor** (редактор)
   - ✅ Може редагувати продукти
   - ✅ Може завантажувати картинки
   - ❌ Не може видаляти користувачів
   - ❌ Не може змінювати налаштування
5. Натисніть **Send invitation**

Співробітник отримає email з посиланням для створення паролю.

## 📱 Як буде працювати

### Для співробітника магазину:
1. Заходить на `https://tanu-xxx.strapiapp.com/admin`
2. Логіниться
3. Бачить список всіх продуктів
4. Може:
   - ✅ Додати новий продукт
   - ✅ Змінити назву/ціну/опис
   - ✅ Завантажити нову картинку
   - ✅ Видалити продукт
   - ✅ Додати нову категорію

### Для сайту:
1. При відкритті меню сайт робить запит до Strapi API
2. Отримує актуальні дані про продукти
3. Якщо Strapi недоступний — показує fallback дані
4. Всі зміни в адмінці одразу з'являються на сайті

## 🎯 Перевірка що все працює

1. **Перевірте API:**
   ```
   https://tanu-xxx.strapiapp.com/api/products?populate=*
   ```
   Має повернути JSON з продуктами

2. **Перевірте frontend:**
   - Відкрийте ваш сайт
   - Натисніть кнопку меню
   - Мають з'явитися продукти з Strapi

3. **Перевірте адмінку:**
   - Зайдіть в Content Manager → Products
   - Змініть ціну будь-якого продукту
   - Оновіть сайт — зміна має з'явитися

## 💡 Поради

- **Завжди робіть Publish** після редагування продукту в Strapi
- **Використовуйте однакові картинки** для всіх продуктів (розмір ~500x500px)
- **Тестуйте зміни** на телефоні перед публікацією
- **Зберігайте токен в безпечному місці** — він потрібен тільки в `index.js`

## 🆘 Якщо щось не працює

**Проблема**: Fallback дані замість Strapi
- **Рішення**: Перевірте що STRAPI_BASE та API_TOKEN правильні

**Проблема**: CORS помилка
- **Рішення**: В адмінці Strapi: Settings → Global Settings → CORS → додайте ваш домен

**Проблема**: 403 Forbidden
- **Рішення**: Перевірте що API токен має права на читання Products та Categories

**Проблема**: Картинки не завантажуються
- **Рішення**: Переконайтесь що завантажили картинки в кожен продукт через адмінку
